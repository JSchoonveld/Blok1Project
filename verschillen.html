Enkele verschillen tussen PHP en Javascript

Php is alleen bedoeld voor de back-end

PHP is nog steeds de meest gebruikte taal voor de back-end van websites. Het populairste content management systeem is Wordpress. Verder wordt er ook veel gebruik gemaakt vaan Joomla en Drupal. elke website maakt gebruik van Javascript voor de front-end. Maar tegenwoordig is het ook mogelijk om de back-end te doen in Javascript door middel van NodeJS.

Omdat PHP een back-end taal is heb je een interpreter nodig om gebruik te maken van PHP op je website. Bij vanilla Javascript is dit niet nodig.

Javascript is asynchronous en PHP niet

Dit houdt in dat je in Javascript bijvoorbeeld een AJAX call kan maken en terwijl de code op een response wacht kan de rest van de code gerund worden. Wanneer PHP code om een IO vraagt pauzeert de code en wacht op de IO om klaar te zijn om verder te gaan.

Javascript kan in je browser gerund worden

Zoals ik hierboven al zei, voor PHP heb je een interpreter nodig. Javascript kan gewoon in de browser worden gerund en met NodeJS ook in de command line

PHP is multi-threaded en JS is single-threaded

PHP kan meerdere taken tegelijkertijd uitvoeren. Bij Javascript is dit niet het geval. Javascript wacht op een event en voert die dan uit.

Combinaties

Js kan gecombineerd worden met HTML, XML en CSS. PHP kan alleen gecombineerd worden met HTML.
